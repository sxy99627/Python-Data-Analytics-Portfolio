## 📊 项目列表
1. **酒店预订需求分析**：探索城市酒店与度假酒店的预订行为规律、季节性趋势、入住需求分布、用户偏爱房型及客户取消预订的关键影响因素。
2. **鸢尾花种类分类建模**：探索不同种类鸢尾花的属性特征。基于鸢尾花的属性数据，分析两种鸢尾花萼片、花瓣的长度和宽度平均值，是否存在显著性差异。
3. **帕默群岛企鹅数据可视化**：探索企鹅物种的体貌特征差异。对帕默群岛上企鹅样本的相关变量进行可视化，分析种类、性别、所在岛屿等因素，与企鹅的身体属性包括体重、嘴峰长度和深度、鳍的长度之间的关系。
4. **Netflix电影数据分析**：挖掘不同流派中的高评分作品演员。整理不同流派影视作品，比如喜剧片、动作片、科幻片中，各演员出演作品的平均IMDB评分，从而挖掘出各个流派中的高评分作品演员。
5. **电商销售数据清洗与分析**：识别销售趋势与异常点，提出运营建议。根据市场销售数据，挖掘畅销产品，以便制定更有效的市场策略来提升营收。
